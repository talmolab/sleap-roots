One `analyze` command composes the whole pipeline on one `.slp`: `Series.load → series_to_inputs → compute_traits → write per-plant CSV → aggregate_by_genotype → write per-genotype CSV → save_plots`. **`adapters.series_to_inputs(series, *, cadence_s, sample_uid, series_name=None, timepoint=None, plate_id=None, genotype=None, treatment=None, r_px=None, run_id=None) -> (CircumnutationInputs, identity_provenance)`** — the single click-free, independently-testable bridge into the pure core, formalizing the former test-only `_load_plate001_inputs()`: `get_tracked_tips()` → **prefix-anchored ASCII `[0-9]+` `track_id` coercion** (strips one leading `"track_"`, then the remainder must be unsigned ASCII digits — raises `ValueError` naming the offender rather than let `int()` silently mis-coerce `"track_-1"`/`"track_1_2"`→`12`/`"track_ 1"`/Unicode digits; `"track_track_1"` raises, not the global-`.replace` corruption to `1`); `plant_id = track_id`; **CSV-as-source / flag-as-override metadata precedence** where a blank value (`None`/empty/whitespace) on EITHER the flag or the CSV cell resolves to absent/`NaN` (so `--genotype ""` or a whitespace cell never forms a degenerate empty-string group); `timepoint` read as the raw `get_metadata` cell and `str`-normalized (object dtype, bypassing the float-coercing `Series.timepoint` property); the six string identity columns forced to `object` dtype even when all-NaN. `--cadence-s` and `--sample-uid` **required** (`--sample-uid` is the stable QR/sample id + metadata-CSV join key, never silently synthesized, per CC-4); `--output-dir` defaults to `./<series-name>_circumnutation/`. **Gated per-genotype aggregation** (`--no-aggregate`): when aggregation is on (default) and `genotype` is unresolved, `analyze` **hard-errors (exit 1, before any output is written)** rather than emit a vacuous `(NaN,NaN,NaN)` per-genotype CSV; the adapter normalizes its raisers to `ValueError`; the command body is wrapped in one `try/except (ValueError, FileNotFoundError) → ClickException` (exit 1, clean one-line message, no traceback); no broad catch-all (genuine bugs surface tracebacks). **`-v`/`-vv`** → WARNING/INFO/DEBUG to stderr (stdout reserved for the result summary), restored in a `finally` so the CLI's global logging config doesn't leak;
