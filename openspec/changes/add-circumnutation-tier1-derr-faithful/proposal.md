## What Changes

- **NEW `nutation.py` module** — `sleap_roots.circumnutation.nutation.compute(trajectory_df: pd.DataFrame, cadence_s: float, coordinate: str = "lateral", constants: Optional[ConstantsT] = None) -> pd.DataFrame` (signature locked here per S8' round-2 reversal — `cadence_s` is an explicit positional parameter mirroring `temporal_cwt.compute_scaleogram`, NOT carried via `trajectory_df.attrs`). Per-track loop mirrors `kinematics.compute` / `qc.compute` precedent. `coordinate=` accepts `{"lateral", "x", "y"}` (default `"lateral"` per CC-7). Output: one row per `(series, sample_uid, plate_id, plant_id, track_id)` with 8 trait columns + 8 identity columns.
- **NEW 8 Tier 1 trait columns** — `T_nutation_median: float` (s), `T_nutation_iqr: float` (s), `A_nutation_envelope_max: float` (px), `band_power_ratio: float` (dimensionless), `noise_floor_estimate: float` (amplitude units), `is_nutating: bool`, `period_residual_vs_derr_reference: float` (dimensionless; S3 round-1 rename from `derr_match_residual`), `cadence_nyquist_ratio: float` (dimensionless; moved from PR #3 per §6.5 deferral). NaN-gating semantics (S4 round-1 + Sci-I3 round-2 reconciliation): when `is_nutating == False`, NaN only 3 strictly biological-meaning-dependent traits (T_nutation_median, T_nutation_iqr, A_nutation_envelope_max); 5 always-populated traits remain finite (is_nutating, band_power_ratio, noise_floor_estimate, period_residual_vs_derr_reference, cadence_nyquist_ratio) so QC investigations can distinguish "no oscillation" from "cadence aliasing" from "ridge-of-noise".
- **NEW `_geometry.project_to_growth_axis_perpendicular(x, y) -> np.ndarray` helper** — estimates growth-axis unit vector from net displacement, constructs perpendicular unit vector via 90° rotation `(-u_g[1], u_g[0])`, projects centered positions. Returns 1D float64 lateral position time series of length `n`. Returns `np.full(n, np.nan)` on zero net displacement (per Architecture-I3 round-1 graceful-NaN policy mirroring kinematics' precedent — NOT `raise`).
- **NEW `_noise.compute_sg_detrended(x, window, polynomial_order) -> np.ndarray` helper (S1 round-1)** — returns the 1D residual `x - scipy.signal.savgol_filter(x, window_length=window, polyorder=polynomial_order, mode='nearest')`. Reuses existing `SG_WINDOW_DETREND = 23` + `SG_DEGREE = 3` constants from PR #2/#3 (no new constants for this helper). Applied to the lateral signal BEFORE CWT/FFT per `preliminary_results §3.4` prescription.
- **NEW `_noise.compute_fourier_noise_floor(x, cadence_s, t_nutation_median_s, factor) -> float` helper** — per CC-8. Returns median `|scipy.fft.rfft(x)|` over frequencies `> factor / t_nutation_median_s`. Returns NaN for `len(x) < 2` or empty out-of-band region.
- **NEW `temporal_cwt.smooth_ridge(ridge_result, window=None, constants=None) -> RidgeResult` primitive (closes #214)** — third composable temporal_cwt primitive alongside `compute_scaleogram` + `extract_ridge`. Median-filters `RidgeResult.periods_s` via `scipy.ndimage.median_filter(...mode='nearest')`; carries through `amplitudes`, `powers`, `in_coi`, `frame_indices` unchanged. Acceptance criterion per #214: `T_nutation_iqr_post_filter < T_nutation_iqr_raw` on ≥5 of 6 Nipponbare plate-001 tracks.
- **6 NEW `_constants.py` defaults (S2'' round-2 final list)** — `RIDGE_CONTINUITY_FILTER_WINDOW: int = 5`; `NOISE_FLOOR_OUT_OF_BAND_FACTOR: float = 5.0` (CC-8 verbatim; GREEN-phase may revisit per S7); `BAND_POWER_BAND_LOW_FACTOR: float = 0.5`; `BAND_POWER_BAND_HIGH_FACTOR: float = 2.0`; `DERR_EXPECTED_PERIOD_S: float = 3333.0`; `TEMPORAL_NYQUIST_RATIO_MAX: float = 0.25` (S2'' round-2 anchor to theory.md §6.5; `NYQUIST_RATIO_MAX` stays at 0.25 for spatial check per PR #9 future use; dimensional separation lives in constant NAMES not VALUES). `_CONSTANTS_VERSION` bumps `4 → 5`. `ConstantsT` gains 6 new fields; `_default_constants_snapshot()` extended (29 → 35 entries).
